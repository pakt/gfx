WebGL remake of Puls by Rrrola
uses glsl sandbox by MrDoob.

run serve.py and visit localhost:8000, or upload to a server --
it's a glsl sandbox requirement :p

to hear the music, you'll need a browser with audio api (webkit
flavour). in chrome, go to about:flags and enable "Experimental APIs".
